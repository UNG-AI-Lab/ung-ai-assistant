# Architecture

This document will outline the system architecture for the UNG AI Assistant.

## Overview

The UNG AI Assistant is a web-based application that helps University of North Georgia students get answers to academic questions.

## Components

- **Frontend**: React application served via Vite
- **Backend**: FastAPI REST API
- **AI Layer**: (TBD) Language model integration for academic Q&A
